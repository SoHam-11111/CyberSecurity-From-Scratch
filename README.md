# 🛡️ Cybersecurity From Scratch

## 📖 About

This repository is my personal cybersecurity knowledge base built while learning cybersecurity from scratch.

The goal is not only to document my own learning but also to create a free resource that anyone starting cybersecurity can use.

Everything here is written in beginner-friendly language with practical examples, diagrams, labs, and interview questions.
